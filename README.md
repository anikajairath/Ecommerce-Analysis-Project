# E-Commerce Sales Performance Analysis

# Overview

This project analyzes e-commerce sales data to evaluate business performance and generate actionable insights. It focuses on revenue, profit, customer behavior, and regional performance to support data-driven decision-making.

# Objective

To understand key business metrics and identify opportunities to improve revenue, optimize product strategy, and enhance customer retention.

# Tools & Technologies

* Python (Pandas)
* MySQL (SQL Queries)
* Power BI (Dashboard)

# Dataset

The dataset contains about 1200 e-commerce transaction records from 2022–2023, including details such as product, category, region, customer, pricing, discount, and payment mode.

# Steps Performed

* Data cleaning and preprocessing using Python (Pandas)
* Handled missing values and removed duplicates
* Converted date columns for time-based analysis
* Created new features like Revenue and Profit
* Performed SQL analysis to extract business insights
* Built interactive Power BI dashboard for visualization

# Key Insights

* Total Revenue: ₹41.36M and Profit: ₹6.80M
* Average Order Value (AOV): ₹34.47K
* Electronics category contributed highest revenue
* West region generated the highest sales
* Clothing category showed lowest performance
* Cash on Delivery (COD) is the most preferred payment method
* Some transactions resulted in negative profit

# Business Impact

This analysis helps businesses identify high-performing products, optimize pricing and discount strategies, and focus on underperforming regions to improve overall profitability.

# Dashboard

Interactive Power BI dashboard showcasing KPIs, revenue trends, category performance, and regional analysis.
